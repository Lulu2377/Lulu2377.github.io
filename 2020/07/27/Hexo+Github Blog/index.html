Hexo+Github Blog

Posted on 2020-07-27

Content

[toc]

Requirements

  1. git
    • Download and Install from Git GUI Clients: https://git-scm.com/downloads
    • $ git -v
  2. nodejs, npm
    • Download and Install from Installer: https://nodejs.org/en/download/
    • $ node -v
    • $ npm -v
  3. Hexo
    • Install Hexo: $ npm install hexo -g
    • Check: $ hexo -v

Github Pages

  1. create an account of github and login.
  2. create a repository named “username.github.io”
  3. open the “Github Pages” in the setting of the above repository and randomly select a theme

Hexo Blog

  1. create the directory of Blog(MyBlog)
  2. $ cd ~/MyBlog
  3. Initialization: $ hexo init
  4. Install the requirements of hexo blog: $ npm install
  5. Generate the Blog: $ hexo g
  6. Start the server: $ hexo s, then you can access the blog in http://localhost:4000/

Link the Hexo and Github

  1. $ git config --global user.name "YourUserName"
    $ git config --global user.email "YourEmailAddress"
  2. $ ssh-keygen -t rsa -C "YourEmailAddress to generate the rsa password. Add the public key into the github. Selecting the personal setting from the Account Setting. Then add new ssh key which you can find it in ~/.ssh/id_rsa.pub to github.
  3. Check: $ ssh -T git@github.com
  4. Modifying the setting file of the Blog named _config.yml.
    $ vim _config.yml
  5. Change the directory of MyBlog. $ vim _config.yml
  6. $ npm install hexo-deployer-git --save
  7. Create a new blog: hexo new hello
  8. Clean the cache. $ hexo clean
  9. Generate the Blog. $ hexo g
  10. Deploy on the Github. $ hexo d
  11. We can find it at: https://YourUserName.github.io/YourUserName.github.io/
